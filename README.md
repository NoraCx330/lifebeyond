# LifeBeyond Deploy

此目录是**自动生成**的部署产物。请通过项目根目录的：
```
node scripts/build-protected.js && node scripts/prepare-deploy.js
```
重新生成。

**不要手动编辑**此目录里的文件，改动会被下次构建覆盖。
